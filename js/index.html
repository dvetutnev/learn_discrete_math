Help

Symbols

  • s generic syllable
  • v vowel
  • V vowel or vowel combination
  • c consonant
  • B consonant combination suitable for beginning a word
  • C consonant combination suitable anywhere in a word
  • i insult
  • m mushy name
  • M mushy name ending
  • D consonant suited for a stupid person's name
  • d syllable suited for a stupid person's name (begins with a vowel)

Syntax

  • <> Symbols between angular brackets generate fragments (above).
  • () Content between parenthesis appears literally.
  • | Generator will select either the left or the right of the bar.
  • ! Capitalize the following component.
  • ~ Reverse the following component.
